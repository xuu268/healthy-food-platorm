# 🍎 健康餐饮AI平台 - 开发项目

## 📋 项目概述
基于商业计划书打造的智能健康餐饮生态平台，专注校园和工厂区市场。

## 🎯 核心功能
1. **AI健康饮食引擎** - 个性化营养建议
2. **创作者生态系统** - 零门槛内容创作变现
3. **智能设备集成** - 健康数据联动
4. **精准流量分发** - AI驱动的商家引流

## 🏗️ 技术架构
- **后端**: Spring Boot + MySQL + Redis
- **前端**: uni-app (微信小程序+H5)
- **AI**: Python + TensorFlow + 计算机视觉
- **部署**: Docker + Kubernetes

## 🚀 快速开始
```bash
# 克隆项目
git clone <repository>

# 启动后端
cd backend && mvn spring-boot:run

# 启动前端
cd frontend && npm run dev:h5

# 启动AI服务
cd ai-engine && python app.py
```
